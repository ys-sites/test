# YS Marketing Solutions

> Helping businesses turn leads into clients through high-performance websites and scaling strategies.

A modern, animated marketing agency website built with **React**, **Vite**, **TailwindCSS v4**, and **Framer Motion**.

## Features

- ✨ Interactive hero with starfield, shooting stars, and mouse-tracking spotlight
- 📊 Animated stats counters with scroll-triggered animations
- 🔍 Card scanner section with Three.js particles and ASCII art effects
- 📱 Fully responsive design optimized for mobile performance
- 📩 Contact form with GoHighLevel/LeadConnector webhook integration
- 🎨 Glassmorphism UI with custom glow effects

## Tech Stack

- **React 19** + TypeScript
- **Vite 6** for blazing-fast builds
- **TailwindCSS v4** with custom theme tokens
- **Framer Motion** for animations
- **Three.js** for particle effects
- **Lucide React** for icons

## Getting Started

### Prerequisites

- Node.js 18+

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will be available at `http://localhost:3000`.

### Build

```bash
npm run build
```

### Environment Variables

Copy `.env.example` to `.env.local` and configure:

- `GEMINI_API_KEY` — Your Gemini AI API key (optional)
- `APP_URL` — The URL where the app is hosted
